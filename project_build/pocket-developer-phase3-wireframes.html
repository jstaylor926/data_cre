Design System ¬∑ Phase 3
Phase 3 introduces a second app operating mode. A toggle in the topbar switches between Development and Data Center mode ‚Äî transforming the map layers, panel tabs, and scoring engine simultaneously. Orange (#f97316) is the Phase 3 system color. The MW slider is the defining interaction ‚Äî drag from 1MW to 500MW, all four infrastructure scores recalculate. CRITICAL environmental flags suppress the composite score entirely and show DISQUALIFIED ‚Äî this cannot be overridden. Voltage-tiered colors key the power infrastructure: red = 500kV, yellow = 230kV, green = 115kV, blue = 69kV.
