Design System · Phase 4
Phase 4 introduces three new surface types that have never existed in this product: (1) a full dashboard at /dashboard with charts and pipeline summary — a separate route from the map entirely; (2) a Kanban pipeline board — the first non-map, non-panel UI pattern; (3) an ML model performance panel with SHAP feature importance. The product also gains a left sidebar for primary navigation across routes. Violet (#a78bfa) is already established as the AI/intelligence accent from Phase 2 — Phase 4 inherits it as the system-wide intelligence color. Pipeline stage cards get colored bottom-stripe identifiers. The ML model is the compounding flywheel — deal outcomes feed training data, SHAP surfaces why the model decided what it decided, retraining shows as a terminal log. Site Watch turns the product proactive for the first time — an alert feed replaces the "open and tap" workflow with "it finds things for you."
